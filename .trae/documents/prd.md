## 1. 产品概述

线下门店分批到货订单分阶段结算系统，解决多批次到货、分批取货、阶梯逾期计费的复杂结算场景。目标用户为线下门店分拣人员和取货顾客，提供从下单、备货、取货到逾期计费的全流程数字化管理。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 分拣员 | 分拣后台入口 | 查看订单、标记备货状态、查看逾期提醒 |
| 顾客 | 取货前端入口 | 查看订单、部分取货、确认取货、查看费用明细 |
| 管理员 | 管理后台入口 | 处理滞留锁定订单、查看对账报表、系统配置 |

### 2.2 功能模块

1. **分拣员后台**：订单列表、批次备货管理、逾期提醒看板
2. **用户取货前端**：订单详情、分批取货操作、费用明细、取货确认
3. **管理后台**：对账流水、滞留订单处理、阶梯计费配置

## 3. 核心流程

### 3.1 订单分批到货与备货流程

用户下单后，商品按到货批次拆分为多条库存记录。分拣人员按批次完成备货标记，系统自动计算各批次入库时间并启动逾期计时。

```mermaid
flowchart TD
    "用户下单" --> "系统创建订单"
    "系统创建订单" --> "商品按批次拆分库存记录"
    "商品按批次拆分库存记录" --> "第一批到货"
    "商品按批次拆分库存记录" --> "第二批到货"
    "商品按批次拆分库存记录" --> "第N批到货"
    "第一批到货" --> "分拣员标记备货完成"
    "第二批到货" --> "分拣员标记备货完成"
    "第N批到货" --> "分拣员标记备货完成"
    "分拣员标记备货完成" --> "通知用户可取货"
    "通知用户可取货" --> "启动逾期计时"
```

### 3.2 分批取货与部分结算流程

用户可提前提取已备货完成的批次商品，系统单独拆分对应货款并生成流水。

```mermaid
flowchart TD
    "用户查看订单" --> "选择可取货商品"
    "选择可取货商品" --> "系统计算本批货款"
    "系统计算本批货款" --> "系统计算本批逾期费用"
    "系统计算本批逾期费用" --> "合并生成本次结算金额"
    "合并生成本次结算金额" --> "用户确认取货"
    "用户确认取货" --> "生成分批取货流水"
    "生成分批取货流水" --> "更新订单状态"
    "更新订单状态" --> "全部取完?"
    "全部取完?" -->|"是"| "订单完成"
    "全部取完?" -->|"否"| "等待下次取货"
```

### 3.3 阶梯逾期计费流程

逾期保管采用阶梯收费：12小时内免费、12-24小时一档单价、超24小时双倍计费，多批次独立计算后合并。

```mermaid
flowchart TD
    "批次到货时间T" --> "计算逾期时长"
    "计算逾期时长" --> "时长 <= 12h?"
    "时长 <= 12h?" -->|"是"| "免费保管"
    "时长 <= 12h?" -->|"否"| "12h < 时长 <= 24h?"
    "12h < 时长 <= 24h?" -->|"是"| "一档单价计费"
    "12h < 时长 <= 24h?" -->|"否"| "超24h双倍计费"
    "一档单价计费" --> "各批次费用汇总"
    "超24h双倍计费" --> "各批次费用汇总"
    "免费保管" --> "各批次费用汇总"
    "各批次费用汇总" --> "生成总服务费"
```


## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青色 (#0F766E) + 琥珀色 (#F59E0B) 作为强调色，体现仓储物流的专业感
- 辅助色：石板灰 (#475569) 用于文本，浅灰 (#F1F5F9) 用于背景
- 按钮：圆角微立体风格，主操作按钮带微妙投影
- 字体：思源黑体 / Noto Sans SC 为主要字体，数字使用等宽字体 JetBrains Mono
- 布局：左侧导航 + 右侧内容区，卡片式信息展示
- 图标：线性图标风格，统一 2px 描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 分拣订单列表 | 搜索筛选栏 | 顶部固定，输入框+下拉筛选，圆角边框 |
| 分拣订单列表 | 订单状态卡片 | 卡片布局，左侧状态色条，右侧进度条 |
| 分拣订单详情 | 批次时间线 | 垂直时间线布局，批次节点带状态图标 |
| 分拣订单详情 | 逾期预警面板 | 琥珀色渐变背景，倒计时数字高亮 |
| 取货首页 | 订单查询 | 居中大输入框，搜索图标动效 |
| 取货订单详情 | 商品批次分组 | 手风琴展开式，批次头显示时间及状态 |
| 取货订单详情 | 费用计算面板 | 阶梯表格，当前阶梯高亮，数字右对齐等宽 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1024px
- 平板适配：侧边栏折叠为图标模式
- 移动端：取货前端适配移动端操作，分拣后台保持桌面端
